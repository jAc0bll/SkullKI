Random Agent

Easy

Bids and plays randomly within the legal move set. Useful as a baseline floor — any intelligent strategy should beat it. Despite being random, it still follows suit-must-follow rules.

Heuristic Agent

Medium

Uses calibrated expected-win weights to bid, then plays with a want-to-win / want-to-lose decision tree. When following a trick, it plays the weakest winning card to conserve strong ones.

MCTS Agent

Strong

Determinization Monte Carlo Tree Search: for each legal action, runs N random game rollouts and picks the action with the highest average outcome. Uses perfect information during rollouts, which is a practical approximation for imperfect-information games. Slower but often stronger on complex decisions.

PPO Agent (Trained AI)

ML

A neural network trained with Maskable PPO via self-play. Takes a 244-dimensional observation (hand, bids, trick history, scores) and outputs a probability distribution over all legal actions. The model improves the longer you train it — use the Settings tab to switch between checkpoints.
